# Project Frontend

## Requirements

- Node.js
- npm or yarn

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the development server:

   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Technologies

- Next.js 14
- TypeScript
- TailwindCSS
- Shadcn/UI (Radix UI integration)
- Class Variance Authority
- Lucide Icons
